### kubectl patch

使用（patch）补丁修改、更新资源的字段。

支持JSON和YAML格式。

### 语法

```
$ patch (-f FILENAME | TYPE NAME) -p PATCH
```

在 patch 命令中，将 `type` 设置为 `merge`：

```shell
kubectl patch deployment patch-demo --type merge --patch "$(cat patch-file-2.yaml)"
```
